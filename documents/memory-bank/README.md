# Memory Bank

Thư mục này chứa thông tin bối cảnh dự án để AI/Dev có thể nhanh chóng nắm được dự án khi quay lại.

## Cách sử dụng

- **AI**: Đọc `PROJECT_CONTEXT.md` trước khi làm việc với dự án
- **Dev**: Tham khảo khi cần refresh context hoặc onboard người mới

## Nội dung

| File | Mô tả |
|------|-------|
| `PROJECT_CONTEXT.md` | Tóm tắt bối cảnh, vấn đề, giải pháp, cấu trúc dự án |
| `PLM_ISSUE_STRUCTURE.md` | Cấu trúc PLM Issue chi tiết: các trường, Comment, gợi ý trích xuất cho RAG |
